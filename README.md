# Vulkan Renderer

Vulkan toy renderer written in C++. Based on previous experience with Vulkan.

## Features

- [ ] OBJ mesh loading
- [ ] Texture mapping
- [ ] ImGui support
- [ ] Virtual camera support
	- [ ] Perspective camera
	- [ ] Orthographic camera
	- [ ] Camera controller
- [ ] Scene loading (glTF2.0)
- [ ] Lights
	- [ ] Sun light (directional, cascaded shadow mapping)
	- [ ] Point lights
- [ ] PBR pipeline
- [ ] Transparency pass
- [ ] Animations
	- [ ] Linear skinning
	- [ ] Blend shapes
- [ ] Volumetric clouds (raymarched)
